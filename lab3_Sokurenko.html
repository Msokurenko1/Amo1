import time
import random
import matplotlib.pyplot as plt
%matplotlib inline

def bubble(nums):
    swapped = True
    a = 0
    while swapped:
        swapped = False
        for i in range(len(nums) - a - 1):
            a += 1
            if nums[i] > nums[i + 1]:
                nums[i], nums[i + 1] = nums[i + 1], nums[i]
                swapped = True

def insertion_sort(nums):
    for j in range(1, len(nums)):
        key = nums[j]
        i = j - 1
        while i >= 0 and nums[i] > key:
            nums[i + 1] = nums[i]
            i -= 1
        nums[i + 1] = key

Контрольні питання¶

  1. Що таке $O-нотація$ і чим вона відрізняється від $\Omega-нотації$?

    $O-нотація$ - це верхня межа, $\Omega-нотації$ - нижня

  2. Яку часову складність має алгоритм сортування методом вставки за найсприятливішої умови, коли масив відсортовано? Відповідь обгрунтувати.

Алгоритм сортування методом вставки за найсприятливішої умови маж складність $O(n)$, бо алгоритм все рівно бере кожен елемент та перевіряє з іншими

  1. Дати визначення ефективного алогоритму.

Ефективний алгоритм - це алгоритм, який працює швидко то ефективно використовує ресурс уя вирішенія конкретної задачі
